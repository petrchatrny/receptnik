Získání receptů
{{ url }}api/recipes?
name={name}&page={page}&api_key={API key}
Parametry
name nepovinné Řetězec, který musí být obsažen v názvu typu jídla.
page nepovinné Udává číslo stránky k načtení.
max_preparation_time nepovinné Zobrazení položek, jejichž doba přípravy je kratší nebo stejná jako hodnota max_preparation_time (v minutách).
newer_than nepovinné Zobrazí recepty, které jsou vytvořeny později, než zadané datum. Datum musí být ve formátu yyyy-mm-dd.
is_vegan nepovinné Možné hodnoty:
  • true Zobrazení položek, které obsahují štítek s danou volbou.
  • false Zobrazení položek, které neobsahují štítek s danou volbou.
is_vegetarian nepovinné Možné hodnoty:
  • true Zobrazení položek, které obsahují štítek s danou volbou.
  • false Zobrazení položek, které neobsahují štítek s danou volbou.
is_gluten_free nepovinné Možné hodnoty:
  • true Zobrazení položek, které obsahují štítek s danou volbou.
  • false Zobrazení položek, které neobsahují štítek s danou volbou.
is_lactose_free nepovinné Možné hodnoty:
  • true Zobrazení položek, které obsahují štítek s danou volbou.
  • false Zobrazení položek, které neobsahují štítek s danou volbou.
difficulty nepovinné Zobrazení položek zadané obtížnosti. Možné hodnoty:
  • 1 - Jednoduché
  • 2 - Střední
  • 3 - Náročné
order_by nepovinné Způsob, jakým mají být záznamy uspořádany podle kritéria "název receptu". Možné hodnoty:
  • name podle názvu
  • rating podle hodnocení
format nepovinné Způsob, v jakém se položky mají zobrazit.
api_key povinné Váš unikátní API klíč, který jste si vygenerovali v naší aplikaci.
