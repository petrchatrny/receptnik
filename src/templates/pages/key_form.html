Vytvořit API klíč

API klíč je identifikátor k ověření uživatele volajícího receptové API. Pomocí tohoto klíče můžete posílat HTTP požadavky na připravené routy našeho API (viz API dokumentace) odkudkoliv.

Maximální počet klíčů na uživatele je 5. Po vytvoření je klíč neaktivní a lze ho aktivovat v tomto webovém rozhraní v uživatelském profilu (rubrika uživatel). Klíč lze případně i deaktivovat.
